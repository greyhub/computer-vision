Hướng dẫn Project 1:
Yêu cầu:
Python 3.6
Numpy
OpenCV > 4.0
1. Đối với trường hợp đồ dùng thông thường:
    - tạo folder "stuff" và để ảnh đồ dùng cần đếm vào trong
    - chạy file stuff.py
    - kết quả số đếm in ở console
    - bấm nút bất kì để chuyển ảnh
2. Đối với trường hợp hạt gạo:
    - tạo folder "rice" và để ảnh hạt gạo cần đếm vào trong
    - chạy file rice.py
    - kết quả số đếm in ở console
    - bấm nút bất kỳ để chuyển ảnh
    - chỉnh sửa dòng 297:
        + Chọn hàm count_general để sử dụng phương pháp chung
        + Chọn hàm count_normal để sử dụng riêng cho ảnh ít thiếu cân bằng sáng
        + Chọn hàm count_his để sử dụng riêng ảnh thiếu cân bằng sáng lớn
